Elder Impulse System

PART 1: MAIN GUIDE

1. Identity

Full Name: Elder Impulse System

Abbreviation: EIS

Category: Trend / Momentum Composite

Creator: Dr. Alexander Elder

One-Sentence Definition: The Elder Impulse System is a bar-coloring technique that combines EMA trend direction with MACD-Histogram momentum to classify each bar as bullish (green), bearish (red), or neutral (blue), acting as a trading "censorship" mechanism.

2. Calculation Logic

Formula

ConditionColorTrading Bias
EMA rising AND MACD-H risingGreenBullish impulse
EMA falling AND MACD-H fallingRedBearish impulse
OtherwiseBlueNeutral/Conflict

Components:

- $EMA_{13}(Close)$ — 13-period Exponential Moving Average

- $MACD\text{-}Histogram = MACD(12,26,9) - Signal(9)$

Classification:


IF EMA[t] > EMA[t-1] AND MACD_Hist[t] > MACD_Hist[t-1]:
    Color = GREEN (+1)
ELSE IF EMA[t] < EMA[t-1] AND MACD_Hist[t] < MACD_Hist[t-1]:
    Color = RED (-1)
ELSE:
    Color = BLUE (0)

Real-World Analogy

Think of the Impulse System like a traffic light for traders. Green light = trend and momentum aligned, go ahead. Red light = trend and momentum aligned against you, stop or go short. Blue light = caution, traffic is conflicted, wait for clarity.

IF Long AND Color == BLUE                  // Momentum conflict
THEN → EXIT or tighten stop (Don't wait for Red)

6. Filter Usage

The Impulse System is a censorship mechanism—it tells you what NOT to do.

RuleApplication
Green barOnly allowed to buy or hold long
Red barOnly allowed to sell or hold short
Blue barNo new positions; reassess

Elder's Trading Rule

- Never sell short when bar is Green

- Never buy long when bar is Red

- Break this rule = fighting the impulse

7. Practical Cautions

Equality Handling: If EMA[t] == EMA[t-1] or MACD-H[t] == MACD-H[t-1], the slope is flat. Code must default to Blue (neutral), not retain previous color.
First Valid Bar: Requires ~35 bars of data (26 for MACD + 9 for signal). First bars produce unreliable colors.

8. Pro Insights

The Blue Light Special: The best entries often occur when bars switch from Green→Blue (pullback) then back to Green. Buying the resumption is safer than chasing long green streaks.

Exit on Blue, Not Red: Dr. Elder recommends exiting longs the moment a Blue bar appears—don't wait for Red. The loss of momentum alignment is your early warning.

Triple Screen Integration: Impulse System is Screen #2 in Elder's Triple Screen. Screen #1 (weekly) sets direction; Screen #2 (daily) uses Impulse for timing.

9. Summary

"The Elder Impulse System is your market traffic light—Green means go, Red means stop, and Blue means the intersection is confused, so proceed with caution."

PART 2: SUPPLEMENTARY GUIDE

1. Core Concept

The Impulse System answers: "Are trend and momentum agreeing right now?"

Markets move when both trend (EMA slope) and momentum (MACD-H slope) point the same direction. When they conflict, the market is "confused." Elder's insight was to visualize this agreement directly on price bars.

2. Statistical Edge

Studies show that trading only in the direction of Impulse color reduces whipsaws by ~30% compared to ignoring momentum alignment.

3. Actionable Strategy: "Color Resumption Entry"


SETUP: String of 3+ Green bars followed by 1-3 Blue bars (pullback)
ENTRY: Bar turns Green again (resumption)
STOP: Below the low of the Blue bars
EXIT: When color turns Blue for 2+ consecutive bars

4. Risks & Limitations

- Flat EMA handling: Must assign Blue, not previous color

- Choppy markets: Excessive Blue bars = no trading edge

- Initialization lag: Need ~35 bars before valid signal

PART 3: ADVANCED SIGNAL REFERENCE

Signal Patterns

Impulse Color Classification

This function determines bar color by checking alignment between EMA trend and MACD-Histogram momentum. Green (+1) indicates both trend and momentum are bullish—the safest environment for long positions. Red (-1) signals both are bearish. Blue (0) indicates conflict where one is rising while the other falls, suggesting caution and no new positions.


def impulse_signal(ema, ema_prev, macd_hist, macd_hist_prev):
    ema_rising = ema > ema_prev
    ema_falling = ema < ema_prev
    hist_rising = macd_hist > macd_hist_prev
    hist_falling = macd_hist < macd_hist_prev
    
    if ema_rising and hist_rising:
        return 1   # GREEN
    elif ema_falling and hist_falling:
        return -1  # RED
    else:
        return 0   # BLUE

Screening Filters


SET Color = ImpulseSystem(13, 12, 26, 9)
FILTER = (Color == GREEN) AND (Color[1] == BLUE)  // Resumption entry

Triple Screen Integration


SCREEN 1 (Weekly): Trend direction (e.g., MACD above zero)
SCREEN 2 (Daily): Impulse System color
SCREEN 3 (Intraday): Entry timing

IF Weekly_Trend == UP AND Daily_Impulse == GREEN:
  ENTER long on intraday pullback
