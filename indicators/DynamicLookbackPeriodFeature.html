Dynamic Lookback Period Feature

PART 1: MAIN GUIDE

1. Identity

Full Name: Dynamic Lookback Period Feature

Abbreviation: DLPF

Category: Adaptive / Meta-Indicator

Creator: Inspired by Kaufman (KAMA) and Chande (VMA)

One-Sentence Definition: The Dynamic Lookback Period Feature automatically adjusts indicator parameters based on real-time market volatility, making indicators faster in high-volatility environments and slower in calm markets.

2. Calculation Logic

Formula

$$N_{dynamic} = N_{max} - \left(\frac{N_{max} - N_{min}}{1}\right) \times \text{NormalizedVolatility}$$

StepFormulaPurpose
1$\sigma = \text{StdDev}(Close, VolatilityPeriod)$Measure current volatility
2$\sigma_{norm} = \frac{\sigma - \sigma_{min}}{\sigma_{max} - \sigma_{min}}$Normalize to 0-1 range
3$N = N_{max} - (N_{max} - N_{min}) \times \sigma_{norm}$Calculate dynamic period
4$N = \text{clamp}(N, N_{min}, N_{max})$Ensure valid bounds

Real-World Analogy

Think of DLPF like adaptive cruise control. In clear highway conditions (low volatility), you set cruise control to maintain steady speed (long lookback). In city traffic (high volatility), you manually adjust constantly (short lookback). DLPF automates this: fast response when needed, smooth cruising when possible.

7. Practical Cautions

Bounds Required: Calculated N must be clamped to [MinPeriod, MaxPeriod] to prevent crashes (N=0) or insufficient data errors.
Jitter Warning: Rapid N changes can cause "teleporting" indicator values and false crossovers.

8. Pro Insights

Meta-Divergence: If price rises but N also rises (volatility dropping), the rally is "complacent" and vulnerable to shock.

The N Plot: Plot the N value itself as a sub-chart. A collapsing N often precedes major price moves.

9. Summary

"Dynamic Lookback is your indicator's autopilot—fast when markets demand attention, smooth when conditions allow patience."

PART 2: SUPPLEMENTARY GUIDE

1. Core Concept

DLPF answers: "Should my indicator be fast or slow right now?"

Fixed-parameter indicators are always a compromise. A 14-RSI is too slow in volatile markets, too fast in calm ones. DLPF bridges this gap by measuring volatility and adjusting N accordingly.

2. Statistical Edge

Adaptive indicators have shown ~15% improvement in risk-adjusted returns over fixed-parameter versions in studies on regime-switching markets.

3. Actionable Strategy: "Volatility Breakout Detector"


MONITOR: N value daily
SIGNAL: N drops by >30% in 3 bars
ACTION: Enter in direction of price breakout
STOP: Opposite side of breakout range

4. Risks & Limitations

- Lookback < 2: Crashes moving average functions

- Lookback > History: Cannot calculate average of more bars than exist

- Over-fitting: Tuning min/max/volatility parameters to past data

5. Further Study

PART 3: ADVANCED SIGNAL REFERENCE

Signal Patterns

Volatility Breakout Detection

This signal identifies when the dynamic lookback period suddenly collapses, indicating a sharp volatility spike. A 30%+ reduction in N typically precedes major price moves as the indicator rapidly adapts to increased market activity. This meta-signal provides early warning of regime change before traditional indicators react.


def volatility_breakout(n_current, n_prev, threshold=0.3):
    collapse = (n_prev - n_current) / n_prev > threshold
    return collapse  # N collapsed = volatility spike = breakout

FILTER = (N_Dynamic < 15) AND (N_Dynamic[5] > 30)  // Recent N collapse
