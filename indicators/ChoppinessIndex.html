PART 1: MAIN GUIDE

1. Identity

Full Name: Choppiness Index

Abbreviation: CHOP

Category: Trend Quality / Fractal Dimension

Creator: E.W. Bill Dreiss (1992)

One-Sentence Definition: The Choppiness Index measures whether the market is trending or choppy (ranging) by comparing price path length to actual displacementâ€”high values mean sideways chop, low values mean a clean trend.

2. Calculation Logic

Formula

$$CHOP = 100 \times \frac{\log_{10}\left(\frac{\sum_{i=1}^{n} ATR_1(i)}{MaxHigh_n - MinLow_n}\right)}{\log_{10}(n)}$$

StepFormulaPurpose
1$TR_i = \max(H_i - L_i,H_i - C_{i-1},L_i - C_{i-1})$True Range for each bar
2$\sum TR = \sum_{i=1}^{n} TR_i$Sum of True Ranges (path length)
3$Range = MaxHigh_n - MinLow_n$Vertical displacement over n bars
4$Ratio = \frac{\sum TR}{Range}$Path efficiency ratio
5$CHOP = 100 \times \frac{\log_{10}(Ratio)}{\log_{10}(n)}$Normalized 0-100 scale

Real-World Analogy

Think of CHOP as a driving efficiency gauge. If you drive 100 miles on a winding mountain road (lots of back-and-forth, high True Range sum) to travel just 10 miles in a straight line (small Range), your efficiency is lowâ€”the market is "choppy." If you drive 15 miles on a highway to travel 14 miles directly, your efficiency is highâ€”the market is "trending."

False Signal Scenarios

1. Using CHOP for Direction

CHOP is 100% directionless. A low CHOP (trending) doesn't tell you if the trend is up or down. You need a separate directional indicator.

2. Early Breakout Entries

CHOP crossing 61.8 doesn't mean breakout happens immediately. Price may still consolidate for several bars.

3. Log Domain Errors

If ATR sum is 0 (flatline price) or Range is 0, the formula produces undefined results (log(0) or division by zero).

Mitigation Strategies

- Always pair with a directional indicator (price breakout, MA cross, MACD)

- Wait for price confirmation after CHOP signals regime change

- Handle edge cases in code (clamp output 0-100, check for zero range)

- Use as a filter, not as a primary entry signal

The V-Turn Warning

When CHOP spikes rapidly from below 38.2 toward 50 or higher while in a trend, it signals that the trend's "smoothness" is gone. The market is becoming choppy even if price is still rising. This is a warning to take profits, not to add positions.

PART 2: SUPPLEMENTARY GUIDE

1. Core Concept

The Choppiness Index measures fractal dimensionâ€”how efficiently price traveled from point A to point B.

High CHOP (> 61.8): Price took a long, winding path. The market is moving in all directionsâ€”up, down, sidewaysâ€”but not making net progress. This is consolidation.

Low CHOP (< 38.2): Price took a direct path. The market is moving efficiently in one direction with minimal back-and-forth. This is a trend.

The formula uses logarithms to normalize the ratio to a 0-100 scale:

- A ratio of 1 (path = displacement) produces CHOP = 0 (perfect trend)

- Higher ratios produce higher CHOP values (more choppy)

Signal Patterns

Regime Classification

CHOP values directly translate to market regime. Values below 38.2 indicate efficient trending movement, while values above 61.8 indicate choppy consolidation. This classification determines whether trend-following or mean-reversion strategies should be employedâ€”a binary decision that dramatically affects trading approach.

Extended Consolidation Detection (Coil Pattern)

Tracking how long CHOP remains above 61.8 identifies "coil" patterns where the market is storing potential energy. Extended periods (30+ bars) of high CHOP historically precede the largest moves, with subsequent 20-bar returns averaging 2.8x normal. This pattern is the highest-value CHOP signal.

Breakout Signal Detection

The transition from consolidation (>61.8) to trending (<61.8) marks the release of stored volatility. This threshold crossing, when combined with price breaking out of its range, provides high-probability breakout confirmation. The signal requires price action confirmationâ€”CHOP alone is directionless.

Trend Exhaustion Detection

Rapid increases in CHOP from trending territory (<38.2) toward neutral (>50) indicate the trend is losing its directional efficiency. This V-turn pattern warns that smooth trending action is deteriorating, signaling time to tighten stops or take profits rather than add to positions.


def chop_regime(chop_value):
    """Classify market regime based on CHOP"""
    if chop_value < 38.2:
        return "STRONG_TREND"
    elif chop_value < 50:
        return "MODERATE_TREND"
    elif chop_value < 61.8:
        return "TRANSITIONAL"
    else:
        return "CONSOLIDATION"

def chop_coil_detector(chop_values, threshold=61.8, min_bars=30):
    """Detect extended consolidation (coil pattern)"""
    above_threshold = sum(1 for v in chop_values[-min_bars:] if v > threshold)
    return above_threshold >= min_bars

def chop_breakout_signal(chop_values, threshold=61.8):
    """Detect transition from consolidation to trend"""
    prev_above = chop_values[-2] > threshold
    curr_below = chop_values[-1] <= threshold
    return prev_above and curr_below

def chop_exhaustion_signal(chop_values):
    """Detect trend losing momentum"""
    prev_trending = chop_values[-2] < 38.2
    curr_rising = chop_values[-1] > 50
    return prev_trending and curr_rising
