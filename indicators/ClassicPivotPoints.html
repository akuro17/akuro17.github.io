ClassicPivotPoints (PP)

PART 1: MAIN GUIDE

1. Identity

Full Name: Classic Pivot Points (Floor Trader Pivots)

Abbreviation: PP, Pivot

Category: Support & Resistance

Creator: Floor Traders (pre-computer era)

One-Sentence Definition: Classic Pivot Points use yesterday's High, Low, and Close to calculate today's central pivot and three levels of support (S1-S3) and resistance (R1-R3)—a self-fulfilling prophecy framework because millions of traders watch these exact levels.

2. Calculation Logic

Formula

Central Pivot:

$$P = \frac{High_{prev} + Low_{prev} + Close_{prev}}{3}$$

Resistance Levels:

$$R1 = 2P - Low_{prev}$$

$$R2 = P + (High_{prev} - Low_{prev})$$

$$R3 = High_{prev} + 2(P - Low_{prev})$$

Support Levels:

$$S1 = 2P - High_{prev}$$

$$S2 = P - (High_{prev} - Low_{prev})$$

$$S3 = Low_{prev} - 2(High_{prev} - P)$$

P(H + L + C) / 3Central Pivot

Real-World Analogy

Think of Pivot Points as a weather forecast for trading levels. Just as meteorologists use yesterday's data to predict today's conditions, traders use yesterday's price action to project today's key levels. The central Pivot is the "expected temperature"—above it is "warmer" (bullish), below is "colder" (bearish).

Opening Range Pivot Play


AT MARKET OPEN:
  Opening_Price = Open
  
IF Opening_Price > Pivot P
  Bias = BULLISH for the day
  Target = R1, then R2
  Stop = Below Pivot

IF Opening_Price < Pivot P
  Bias = BEARISH for the day
  Target = S1, then S2
  Stop = Above Pivot

Pivot Bounce Entry


IF Price approaches Pivot from above
AND Pivot acts as support (price bounces)
THEN → BUY at bounce
Stop = Below Pivot
Target = R1

IF Price approaches Pivot from below
AND Pivot acts as resistance (price rejects)
THEN → SELL at rejection
Stop = Above Pivot
Target = S1

Breakout Entry


IF Price breaks above R1 with conviction
(Closes above R1, not just wicks)
THEN → BUY breakout
Stop = Below R1 (now support)
Target = R2

IF Price breaks below S1 with conviction
THEN → SELL breakdown
Stop = Above S1 (now resistance)
Target = S2

7. Practical Cautions

Lag Characteristics

ComponentNotes

Level CalculationStatic for entire session
UpdatesOnly at session change

False Signal Scenarios

1. Timeframe Mismatch

Using previous 5-minute bar instead of previous day's candle. This is a logic error—pivots are always based on higher timeframe data.

2. IPO/First Day

No previous session exists. Indicator returns null.

3. Gap through Levels

Large gap opens can cause price to start above R1 or below S1, making those levels meaningless for the session.

Mitigation Strategies

- Always verify aggregation period — daily pivots for intraday, weekly for daily

- Handle missing data — return null on first trading day

- Check for gaps — if open gaps past R1/S1, use R2/S2 as primary levels

- Wait for level tests — don't assume levels work until proven

8. Pro Insights

Pivot Range Width Indicates Day Type

Compare today's R1–S1 distance to yesterday's:

- Narrow Range (R1–S1 < yesterday): Expect a breakout (Trending day likely)

- Wide Range (R1–S1 > yesterday): Expect mean reversion (Price stays inside R1/S1)

This simple comparison predicts day type before the market opens.

The "Virgin" Pivot

If price fails to touch the Central Pivot (P) for the entire day, it indicates an incredibly strong trend. That untouched pivot level becomes a magnet for future sessions—price will likely revisit it within the next 1-3 days.

R1–S1 Is the "Normal" Zone

In ranging days, 70%+ of price action occurs between R1 and S1. Professional intraday traders fade moves to these levels rather than chasing beyond them.

The Pivot as Bull/Bear Dividing Line

The simplest use: above P = bullish, below P = bearish. This applies throughout the day. If price crosses above P mid-session, bias shifts bullish; crossing below shifts bearish.

9. Summary

"Classic Pivot Points are self-fulfilling support/resistance levels because millions of traders watch them. The central Pivot divides bulls from bears, the R1-S1 zone contains normal day action, and the width of that zone predicts whether the day will trend or range."

PART 2: SUPPLEMENTARY GUIDE

1. Core Concept

Pivot Points convert yesterday's trading range into today's support/resistance framework.

Why they work: Self-fulfilling prophecy. Millions of traders (especially institutions and floor traders) watch the same levels, creating actual buying/selling pressure at those prices.

The hierarchical structure:

- P (Pivot): The equilibrium point

- R1/S1: First-tier levels, most frequently tested

- R2/S2: Second-tier, significant support/resistance

- R3/S3: Extreme levels, rarely reached

2. Statistical Edge

S1-R1 Containment

On average, approximately 70% of trading sessions see price contained between S1 and R1. This makes fade strategies at these levels statistically sound.

Pivot Touch Rate

The central Pivot is touched in roughly 65% of trading sessions, making it a high-probability target for mean reversion.

3. Actionable Strategy

Strategy Name: "Opening Gap + Pivot Bias"

Signal Patterns

Pivot Bias Determination

The relationship between opening price and the central pivot establishes the day's directional bias. Opening above the pivot suggests bullish sentiment for the session, while opening below suggests bearish. This simple comparison is the foundation of floor trader strategies and has self-fulfilling power because millions of traders use these exact levels.

Pivot Bounce Detection

Identifying when price approaches and reacts to pivot levels enables high-probability bounce trades. The central pivot is touched in approximately 65% of sessions, making it a reliable target for mean reversion trades. The tolerance parameter accounts for slight overshoots while maintaining signal precision.

Breakout Through Pivot Levels

Clean breakouts through R1/S1 levels signal trend continuation days. These breakouts are most reliable when accompanied by above-average volume and when the day's R1-S1 range is narrower than normal—suggesting a trending day rather than a range day.

Day Type Prediction

Comparing today's pivot range width to yesterday's predicts whether the session will trend or range. Narrower ranges suggest stored energy and breakout potential, while wider ranges suggest mean reversion to the pivot. This simple comparison has predictive value before the market even opens.


def pivot_bias(open_price, pivot):
    """Determine daily bias from opening price vs pivot"""
    if open_price > pivot:
        return "BULLISH"
    elif open_price < pivot:
        return "BEARISH"
    else:
        return "NEUTRAL"

def pivot_bounce(price, level, tolerance_pct=0.002):
    """Detect bounce at pivot level"""
    near_level = abs(price - level) / level < tolerance_pct
    return near_level

def pivot_breakout(price, prev_price, level):
    """Detect breakout through level"""
    cross_above = prev_price <= level and price > level
    cross_below = prev_price >= level and price < level
    return cross_above, cross_below

def day_type_prediction(r1, s1, prev_r1, prev_s1):
    """Predict day type from pivot width comparison"""
    today_width = r1 - s1
    yesterday_width = prev_r1 - prev_s1
    if today_width < yesterday_width * 0.9:
        return "TRENDING_EXPECTED"
    elif today_width > yesterday_width * 1.1:
        return "RANGING_EXPECTED"
    else:
        return "NORMAL"
